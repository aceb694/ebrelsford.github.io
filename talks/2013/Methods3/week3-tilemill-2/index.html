CartoCSS

We have a layer named bids. We can style the features in it like so:

    #bids {
        line-color: #ae8;
        line-width: 0.5;
        polygon-opacity: 1;
        polygon-fill: #ae8;
    }

Use variables to remove redundancy and improve flexibility:

    @bidcolor: #ae8;
    #bids {
        line-color: @bidcolor;
        line-width: 0.5;
        polygon-opacity: 1;
        polygon-fill: @bidcolor;
    }

Advanced selectors

    #my-points {
      marker-height: 60;
      marker-width: 60;
      marker-fill: #f45;
      marker-line-color: #813;
      marker-line-width: 15;
      marker-allow-overlap: true;
    }
    #my-points[] {
      marker-height: 90;
      marker-width: 90;
    }

Vary styling based on the map's zoom level

    #my-points {
      marker-height: 60;
      marker-width: 60;
      marker-fill: #f45;
      marker-line-color: #813;
      marker-line-width: 15;
      marker-allow-overlap: true;
    }
    #my-points[zoom = 14] {
      marker-height: 90;
      marker-width: 90;
    }

Vary styling based on the features' attributes

    #my-points {
      marker-height: 60;
      marker-width: 60;
      marker-fill: #f45;
      marker-line-color: #813;
      marker-line-width: 15;
      marker-allow-overlap: true;
    }
    #my-points[population > 300000] {
      marker-height: 90;
      marker-width: 90;
    }
