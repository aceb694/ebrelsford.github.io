Making BBLs with OpenRefine

https://data.cityofnewyork.us/Social-Services/Bronx-Vacant-Lots-Cleaned/nugr-jv72
MapPLUTO has BBLs

BBL = borough, block, and lot number

ABBBBBCCCC

  • A: borough number
  • B: block number
  • C: lot number

3015510102

  • 3: borough number (Brooklyn)

3015510102

  • 3: borough number (Brooklyn)
  • 01551: block number

3015510102

  • 3: borough number (Brooklyn)
  • 01551: block number
  • 0102: lot number

Borough numbers:

  • 1: Manhattan
  • 2: Bronx
  • 3: Brooklyn
  • 4: Queens
  • 5: Staten Island

One way to fix this: OpenRefine 

"0000"[0, 4-value.length()] + value

via OpenRefine recipes 

'2' + cells['Block'].value + cells['Lot'].value
