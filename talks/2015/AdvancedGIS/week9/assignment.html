Part 1: JavaScript

You're not expected to fully know JavaScript for this course. However, being able to find answers to JavaScript questions and understand the fundamentals will make the next two weeks much easier for you. With that in mind, explore the introductory lessons from one of the following JavaScript courses:

  • Khan Academy
  • Codecademy
  • Eloquent JavaScript
