Make a dynamic page using CartoDB.js

Using CartoDB.js create a webpage that:

  1. Loads a CartoDB visualization. Do not embed the visualization using an iframe.

  2. Performs an operation when a feature is clicked or moused over (featureClick, featureOver).

  3. Incorporates at least one API other than the CartoDB SQL API. You are free to use the SQL API, but please use something else in addition to it.

    The API can either be called dynamically or it can be the source for your data (eg, you might use the API and OpenRefine to download the data, then upload it to CartoDB).

  4. Post the page somewhere publicly where it can be used. GitHub Pages is preferred.
