Choose you own adventure

Choose to either work more with Mapbox Studio or with templates.

Choice 1: Mapbox Studio

Make and upload a style with Mapbox Studio that you have customized significantly.

  1. Start with a Mapbox Studio style.
  2. Customize the style in some non-trivial way. That is, do not simply change a few colors, but please change multiple aspects of the style. Make it your own.
  3. Commit the style to a GitHub repository.
  4. Upload the style to mapbox.com and use it in a Leaflet or CartoDB map.

Choice 2: Templates

Use templates to display data as HTML in CartoDB or Leaflet.

  1. Add functionality to a Leaflet or CartoDB map that shows data through Mustache (or other) templates.
  2. If you're using CartoDB, please customize the infowindows significantly as they're already using Mustache templates.
  3. Commit your finished page to GitHub and ensure that it's visible on GitHub Pages.
