In-class Excercises

Part 1: JavaScript refresher

Recall your web development skills through this exercise. Write this code in a code editor on your computer or use an online code editor such a JS Bin.

  1. Create an HTML page that contains at least three elements: one that is a form element (input, textarea, or select), and two elements that you plan on styling with CSS (links or divs will do).
  2. Style your page using CSS.
  3. Add JavaScript to your page such that when the form element changes in a particular way one of the non-form elements' CSS properties change, and when it changes in another way the other non-form element's CSS changes.
  4. Post your page to Gist or a GitHub repository.

Part 2: CartoDB

Let's get familiar with CartoDB's interface.

  1. Create a CartoDB account using an email address ending in .edu if you have one.
  2. Download some data, such as the USGS's earthquake data or AirBnB listings via Inside AirBnB (for the latter you want the fourth file down, nyc_listings_20150101184336.csv).
  3. Upload the data to CartoDB and explore its visualization wizards.
  4. Do the same with a shapefile, for example City Council Districts on the city's open data portal. Shapefiles have to be zipped to be uploaded to CartoDB.
  5. Edit the infowindows for your visualization.
  6. Publish your visualization and get a public link for it.
