In-class Excercises: PostGIS

The PostGIS documentation will likely be helpful for the following.

Use this week's data for these exercises.

Part 1: Spatial Joins

  1. Join the bikelanes and city_council_districts tables using the spatial relationship function of your choice.
  2. Count the number of bike lanes in each city council district.
  3. Measure the total length of the bike lanes in each city council district.

Part 2: Schools

  1. Join the public_school_locations and city_council_districts.
  2. Count the number of schools in each city council district.
  3. Find the average SAT score of each city council district.

Part 3: SQL like Turf

Attempt to recreate a few functions that Turf provides using only SQL. Think max or convex.
