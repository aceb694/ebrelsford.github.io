In-class Excercises: Git and GitHub

Part 1: Git Branches

  1. Create a branch in the Git repository that you created for the previous assignment: 
    git checkout -b branch_name
  2. Make some changes on the branch and commit them: 
    git add files_with_changes
git commit -m"commit_message"
  3. Push your branch to GitHub: 
    git push origin branch_name

Part 2: Making pull requests

  1. Ask a fellow classmate for a GitHub repository you can make a change to.
  2. Fork the repository in GitHub to your own account.
  3. Clone the repository to your computer.
  4. Create a branch for the changes you're about to make.
  5. Make some changes to the repository in your new branch.
  6. Commit the changes.
  7. Push the branch to GitHub.
  8. Create a pull request back to your classmate's repository.
  9. Experiment with leaving comments on pull requests and commits.
  10. The classmate doesn't have to accept the changes if they're not useful.

Part 3: geojson.io

  1. Fork this repository.
  2. Open geojson.io and log in with your GitHub account.
  3. Go to Open > GitHub and open 596acres-02-18-2014-queens.geojson from the repository that you forked.
  4. Make some changes to the file. Add some geometries, delete some.
  5. Click Save to commit your changes. Do not click any of the items in the menu that opens when you mouseover Save.
  6. View the changes in your GitHub account's repository.
  7. If you have time, create a repository with your own data and do the same.
