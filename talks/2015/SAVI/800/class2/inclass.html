Advanced Interactive Web Mapping, Programming, and Design

In-class Excercises

Part 1: SQL

  1. Open a table in CartoDB.
  2. Open the SQL cheatsheet.
  3. Execute a few SQL queries using the patterns outlined in the cheatsheet:
    • filter data
    • count data
    • group data
    Keep track of these in a text file.

Part 2: CartoDB's SQL API

  1. Continue using the CartoDB table and SQL queries from Part 1.
  2. Run an SQL query using CartoDB's SQL API. It should be of the form: 
    https://{account}.cartodb.com/api/v2/sql?q={sql}
    Go to this URL using your browser.
  3. Select all of the response you get back and copy it.
  4. Open your JavaScript console, type the following: 
    response = '
    Paste the copied text, add a ' to close the quotes, and hit [enter].
  5. Explore the response using the console. Type response, hit [enter] and see the results.
  6. The data is stored in the key rows. View the contents of this by entering response.rows in your console and pressing [enter].
  7. rows is an array. In the console, examine the first row.
  8. Can you find the data within the response's rows?
  9. Do the same as above entirely in JavaScript. Use jQuery's $.getJSON() function to fetch the URL and log out some of the data.
