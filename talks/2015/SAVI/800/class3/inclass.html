In-class Excercises: APIs

Part 1: Nominatim

  1. Check out the documentation for Nominatim.
  2. Starting with the code in this JS Bin, make the changes in the next few steps.
  3. Call Nominatim using the query as entered in the search input using $.getJSON().
  4. Once you get a response, try logging out the latitude and longitude of the first result.
  5. Pan and zoom the Leaflet map to the point (latitude and longitude) found in the previous step by calling map.setView().
  6. Wouldn't it be nice to see an outline of the building or place you searched for? Nominatim can help with that! Look for a parameter called polygon_geojson in the Nominatim documentation and add it to your requests.
  7. Find the GeoJSON in the first result in the response. It's not always going to show up (eg, if Nominatim can't find a specific polygon for a query), so use "Higgins Hall Brooklyn" or "Union Square Park" as test queries.
  8. Finally, add that GeoJSON to the GeoJSON layer (geoJsonLayer in the jsbin). You should be able to call geoJsonLayer.addData() with the GeoJSON from the response.

Part 2: Streetview image API

  1. Using Google's Street View Image API, attempt to load an image in your browser using the URL bar—not in code. If you need a latitude and longitude to start with, try Higgins Hall: 
    40.68802005, -73.9642238450178
    .
  2. Continue with your code from Part 1, and we'll add a streetview image of the search result using the Street View Image API.
  3. Add an img tag to the HTML. In the map div is a good place. For now it won't have a src attribute, so leave it out. Give it a class so you can style it and reference it in your JavaScript.
  4. Add some CSS to make the image show up on top of the map. This will make it appear in the bottom left of the map: 
    .streetview {
  display: none;
  position: absolute;
  bottom: 10px;
  left: 10px;
  width: 200px;
  height: 200px;
  z-index: 5000;
}
    (Update the class name as necessary.) Note that we're starting with display: none as there will be nothing to show at first.
  5. Now add a new function in your JavaScript that take two parameters, lat and lng. In the function, create an image url using the Street View Image API and set your image's src attribute to it using jQuery's attr(). Show the image using show().
  6. Call this new function from the code that calls Nominatim and loads the result.

Part 3: OpenRefine

  1. Install OpenRefine if you haven't already.
  2. Using your browser's developer tools, find the url used to load the markers on this HabitatMap.
  3. Copy the response from developer tools and paste into OpenRefine (Create Project > Clipboard), then export the data as a CSV.
